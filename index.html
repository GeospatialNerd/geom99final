Monitoring Biomass in Lake Simcoe

A web based GIS solution for residents to monitor genenral lake health.

Introduction

This website will take you through the development and learning process undertaken by the GLAM Solutions team for creating a web based solution for monitoring the health of Lake Simcoe using biomass as an indicator.

Problem Statement

Ideal

Residents surrounding Lake Simcoe are concerned about the general health of the lake. Our team will investigate changes in levels of biomass in Lake Simcoe for 2008 and 2013 to gauge the health of the lake

Reality

As far as we're aware, there's currently no web-based system in place that displays biomass data over time, so interested parties are lacking a consistent, shared source of information.

Consequences

Lack of information of general lake health indicated by biomass does not provide opportunities to identify areas of concern or promote conservation efforts.

Proposal

Implement a map-centric web app that displays biomass data for Lake Simcoe in an accessible manner.

Technology Used

ArcGIS Technology (ArcGIS Pro, ArcGIS Dashboards, ArcGIS Online) and VertiGIS Studio

Project Process / Software Used

ArcGIS Dashboards

Our team started this learning process with the use of ArcGIS Dashboards. It's a useful tool for generating a simple one page application that contains spatial information and allows for the inclusion of charts and numeric reporting.
